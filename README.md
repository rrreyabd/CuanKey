# CuanKey

## Project Overview

**CuanKey** is a mobile application designed to help users track their income, expenses, savings, and financial goals in a simple and intuitive way. This app provides real-time insights into spending habits, budgeting tools, and financial reporting, enabling users to make informed financial decisions.

## Features
- **User Authentication** <br> 
Easily sign up and log in to the app with a secure authentication process.
- **User Profile** <br> 
Manage personal details in-app.
- **Push Notifications** <br> 
Notify users of important events.
- **Account Management** <br> 
Switch between multiple savings or bank accounts effortlessly within the app.
- **Income and Expense Tracking** <br> 
Log your income and expenses by category for better financial visibility.
- **Recurring Income and Expenses** <br> 
Automatically record recurring transactions like monthly salary or regular bills.
- **Transaction Notes** <br> 
Add detailed notes to your transactions for better clarity and future reference.
- **Transaction Filtering** <br> 
Easily filter your transactions based on date, category, or account to find specific entries.
- **Transaction Reports** <br> 
Generate detailed reports on your income and expenses, providing insightful financial summaries.
- **Goals and Savings Plans** <br> 
Set savings targets and track your progress toward financial goals.
- **Financial Dashboard** <br> 
View a comprehensive overview of your financial status, including expenses, income, and savings in one intuitive dashboard.

## Tech Stack

### Frontend (Mobile)
- **Language**: JavaScript
- **Framework**: React Native
- **Tools**: Expo for streamlined React Native development
- **Networking**: Fetch API for API requests
- **Local State Management**: React Context API

### Backend (API)
- **Language**: PHP
- **Framework**: Laravel 11
- **Database**: SQLite
- **API**: RESTful API using Laravel Sanctum for Authentication

## Installation

1. **Clone the Repository**

   Clone this project to your local machine.

2. **Install Dependencies**

   Install the necessary dependencies:
   ```bash
   npm install

3. **Start an Expo development server**

   Start an Expo development server and it will display all commands you need

   ```bash
   npx expo start

### Mobile Emulator

4. **Run the Application on your Android Emulator**

   Press `a` after `npx expo start` command.

   Or you can use a feature that Expo offers, which allows you to run your project wirelessly via Expo Go.

### Expo Go (Android/iOS)

4. **Download Expo Go**

   Download Expo Go from Google Play Store or Apple App Store.

5. **Connect Your Mobile Device and Desktop to the Same Network**

   Make sure both your **desktop** and **mobile device** are connected to the **same Wi-Fi network** for proper communication.

6. **Open Expo Go and Scan the QR Code (or Enter URL Manually)**

- **Option 1: Scan the QR Code**
  - Once the development server is running, you'll see a QR code displayed in the terminal or Expo DevTools in your browser.
  - Open the **Expo Go** app on your mobile device.
  - Use the built-in QR scanner in Expo Go to scan the QR code displayed on your screen.

- **Option 2: Enter the URL Manually**
  - If you can't scan the QR code, you can enter the URL manually.
  - In your terminal or Expo DevTools, you'll see a URL similar to:

    ```
    exp://192.168.0.1:19000
    ```

  - Open the **Expo Go** app on your device.
  - Tap the **"Enter URL manually"** option.
  - Type in the URL shown in your terminal or DevTools (e.g., `exp://192.168.0.1:19000`) and tap **"Connect"**.
  - Your app will then load on your device.
